# Starling Desktop

Native GUI client for [Starling](https://forgejo.hearthhome.lol/Saltfault/Starling) —
a federated peer-to-peer communications platform.

## Status

🚧 Not yet started. Will provide text, voice, and video calls through a
native desktop GUI.

## Dependencies

- [starling-server](https://forgejo.hearthhome.lol/Saltfault/Starling-Server) — shared protocol library
